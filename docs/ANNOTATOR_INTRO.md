# 标注者上手指南（UI 复核）

面向日常标注人员的简明说明，目标是快速、稳定地完成 P1/P2 双人框复核。

## 你要做什么

逐帧确认并修正 P1（绿衣服）与 P2（灰衣服）的人体框，保证同一视频内人物身份保持一致。

## 如何开始

1. 打开标注页面 `http://<host>:10086`（具体地址由项目负责人提供）。
2. 在顶部输入你的标注员 ID（首次会提示填写）。
3. 等待画布加载后开始标注。

## 界面区域速览

- 左侧：历史记录列表，可折叠并进入编辑模式。
- 中间：视频帧画布与操作提示。
- 右侧：P1/P2 控制面板与参数输入。
- 顶部：进度条与主要操作按钮（跳过、提交）。

## 每帧标注流程

1. 先检查 AI 框是否可用。
2. 对 P1/P2 分别选择操作：应用 AI 框、手绘新框、或标记不存在。
3. 通过拖拽/缩放微调框位置与大小。
4. 点击“提交并下一帧”进入下一帧；如果不适合标注，使用“跳过当前帧”。

## 关键操作提示

- “应用 AI 框”：快速采用 AI 结果后再微调。
- “绘制新框”：按住拖拽绘制，松开完成。
- “不存在”：该人物在此帧不可见时使用。
- 历史记录：点击条目可进入编辑模式并保存修改。

## 质量要点

- 框尽量紧贴人物可见区域，避免过大或过小。
- 保持 P1/P2 身份一致，避免前后帧互换。
- 遮挡时以可见部分为准，不随意扩大到不可见区域。
- 不确定时优先保证框稳定与一致，再提交。

## 帧分配/采样逻辑（了解即可）

你打开 UI 或点击“下一帧”时，后端会从全局帧池里**随机分配**一帧，规则如下：

1. 优先从 `annotation_count < 3` 的帧集合中随机抽取。
2. 若全部 `>= 3`，则从全局最小 `annotation_count` 的帧集合中随机抽取。
3. 计数只会在“提交并下一帧”成功后 `+1`，单纯点“下一帧”不会改变计数。
4. 随机性由服务启动参数 `--seed` 决定，因此在相同状态下可复现。
5. 分配不做排他锁，多人同时标注可能会抽到同一帧（系统允许重复标注）。

## 自动推荐 AI 框（D 阶段）

当你领取新帧时，系统会基于历史标注**自动选择**部分 AI 框作为默认 P1/P2：

1. 系统维护统计表：`(video_stem, ai_track_id) -> {p1_count, p2_count}`。
2. 每次提交后，只要该 slot 不是 `absent` 且有 `ai_track_id`，就会累加对应计数。
3. 领取新帧时，对该帧的每个 AI `track_id` 查询统计：
4. 若 `p1_count > p2_count` 则推荐 P1；若反过来则推荐 P2；相等不推荐。
5. UI 只在**你未操作该 slot**时自动应用推荐，避免覆盖你的手动修改。

## 遇到问题怎么办

- 画布加载慢：稍等或尝试刷新页面。
- 操作疑问：查看顶部提示或联系项目负责人。
